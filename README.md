# CopyrightPass (CRP)

**CRP is a digital rights protocol for IP licensing, verification, governance, and on-chain asset records.**

It connects real-world intellectual property (IP) with on-chain verifiable rights through PLR / PPR standards and DAO governance.

## 📦 Core Modules

### 1. CRP Token System
- ERC-20 token on BNB Chain
- Governance + utility asset of the ecosystem

### 2. PLR / PPR Rights System
- PLR: Licensing Rights
- PPR: Production Rights
- On-chain proof + off-chain legal verification

### 3. DAO Governance
- VotingEscrow (veCRP)
- Proposal-based governance
- Timelock execution

### 4. IP Verification Layer
- ECDSA signature verification
- Approved certification entities (e.g. HKDAA)
- Merkle Tree asset records

## 🧠 What is CRP?

CRP is not just a token.

It is a **rights coordination protocol** that enables:

- IP assets to be verified on-chain
- Ownership to be structured into PLR/PPR layers
- Governance to control IP lifecycle
- Real-world assets to connect with blockchain systems
