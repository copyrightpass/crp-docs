## Token Description

CopyrightPass (CRP) is a digital tool token designed for IP licensing, PLR/PPR rights verification, DAO governance, and Merkle Tree asset records.

CRP is intended to support verifiable IP authorization and production-right verification. It does not represent equity, profit distribution, revenue sharing, or royalty income rights.
